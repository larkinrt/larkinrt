About Me

I am currently a PhD student in the Department of Second Language Studies at the University of Hawaiʻi at Mānoa. You can check out my CV for detials of my work in Applied Linguistics. I do other things too, like climbing and writing bad poetry. My grandparents on both sides were part of the Hillbilly Highway.

About This Website

This website was built using Hexo! Hexo makes it easy to build static sites such as this in markdown. Please see the documentation for more info. The theme used is Cactus. Previously I hosted this page using the University of Hawaiʻi’s free SFTP hosting, however I couldn’t get hexo to automatically deploy using it. I’m not sure if this was an error on my part or a limitation of UH’s hosting service, or a problem with hexo. At any rate, I am now hosting it on github pages.
